== Description ==

Selleradise Widgets is a plugin for the Selleradise theme. It contains Elementor widgets which are necessary to create a landing page shown in the Demo of Selleradise theme.

## Widgets Included
* Hero
* Promo Cards
* Categories
* Products
* Sale Countdown
* CTA
* Incentives
* Features
* FAQs
* Tabs
